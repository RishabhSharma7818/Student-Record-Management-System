# Student Record Management System

This is a simple Java console-based project to manage student records.  
It allows adding, viewing, updating, and deleting student information.

## Features
- Add a new student (ID, Name, Marks)
- View all students
- Update an existing student
- Delete a student using ID
- Uses ArrayList for storing student objects
- Uses OOP concepts like classes, objects, constructor, getters and setters

## Technologies Used
- Java
- Eclipse
